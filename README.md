## 🚀 快速开始

### 1️⃣ 下载插件
- 点击页面右上角的绿色 "Code" 按钮
- 选择 "Download ZIP" 下载完整代码包
- 解压到本地文件夹

### 2️⃣ 安装插件
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择解压后的项目文件夹

### 3️⃣ 开始使用
- 在 Chrome 工具栏中找到 "Pump项目方一键分析" 插件
- 点击插件图标打开侧边栏
- 按照界面指引完成设置
- 开始分析代币项目方信息
